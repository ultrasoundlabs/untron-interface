  ## What it does

  Returns the effective realtor configuration for the caller (and whether they’re allowed).

  ## Request

  :::code-group

  ```bash [cURL]
  curl -sS 'https://api.untron.finance/v3/realtor'
  ```

  :::

  ## Response (200)

  {
    "allowed": true,
    "realtor_address": "T…",
    "user": "…",

    "default_duration_seconds": 86400,

    "arbitrary_lessee_flat_fee": false,
