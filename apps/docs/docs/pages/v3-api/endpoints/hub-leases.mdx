  ## What it does

  Returns hub-indexed lease rows (current registry + some metadata).

  ## Request

  :::code-group

  ```bash [cURL]
  curl -sS 'https://api.untron.finance/v3/hub_leases?order=lease_id.desc&limit=10'
  ```

  :::
